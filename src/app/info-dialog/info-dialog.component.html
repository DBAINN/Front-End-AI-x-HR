Istruzioni per il caricamento dei file excel

Prima di procedere alla creazione e successivo caricamento dello ZIP, ricorda di rinominare i file excel facendo precedere al nome degli stessi, il numero identificativo scelto in accordo alla tipologia di contenuto:

  • 1 => Risorse attive
  • 2 => Presenze Fiori
  • 3 => Diarie
  • 4 => Richieste Ferie/Permessi/ROL (mytime)
  • 5 => Riepilogo Note Spese
  • 6 => Rendicontazione Ferie Permessi (mytime)
  • 7 => Dipendenti per Sede

Ad esempio se i file relativi rispettivamente alle "Risorse attive" ed alle Presenze estratte da Fiori sono denominati "Risorse attive PRO Aprile 2024" e "Presenze Fiori PRO 01/04/2024" prima di procedere al loro caricamento nel file zip dovranno essere rinominati "1 Risorse attive PRO Aprile 2024" e "2 Presenze Fiori PRO 01/04/2024"
